    PsyanimPreyPrefab

    Extends:

    PsyanimEntityPrefab → PsyanimPreyPrefab

    Prefab for creating a Prey Agent.

    A Prey Agent will wander until the target comes within its field-of-view and within the panic distance, at which point it will begin to flee from the target in a direction offset by the subtlety parameter.

    A Prey Agent has the following components:

    PsyanimVehicle, PsyanimFleeBehavior, PsyanimSeekBehavior, PsyanimWanderBehavior, PsyanimFOVSensor, PsyanimBasicPreyBehavior, PsyanimPreyAgent
